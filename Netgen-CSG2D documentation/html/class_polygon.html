TopoDS_Shape Polygon::MakePolygon ( )

Creates a closed polygon represented as a bounded surface in the 3D space.

It constructs a polygon by adding all the points in arrayOfPoints, adds it to the wire and eventually, creates a face resulted from taking the closed area determined by the wire on a bounded surface. It returns the face resulted.

Returns:
The face for the polygon.
