TopoDS_Shape CSGScanner::ParseCSG ( istream &  istr)

The csg2d file parser.

Main parsing function for CSG2D geometries.

This is the main function which scans the whole file and acts according the information read.

Parameters:
istrA istream & argument.

This method checks for the 'algebraic2d' keyword of the CSG2D input files format. It defines planefigures, TLOs, flags for the TLOs and constructs the final compound shape.

Parameters:
istrAn istream argument. The CSG2D input file to be parsed.
Returns:
the final compound shape.
